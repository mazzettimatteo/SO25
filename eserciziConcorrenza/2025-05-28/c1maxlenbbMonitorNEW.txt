Esercizio c.1: Scrivere il monitor maxlenbb che fornisce le seguenti procedure entry:
void maxlenbb.add(int bufnum, objtype obj)
objtype maxlenbb.get(void)
Il monitor gestisce N buffer ognuno dei quali contiene NELEM elementi.
La funzione add inserisce obj come elemento nel buffer indicato da bufnum.
La funzione get cerca il buffer che contiene più elementi e da questo toglie in ordine FIFO un elemento e lo
restituisce.
In caso più buffer abbiano lo stesso numero massimo di elementi viene scelto il buffer di indice minore.

array of (buffer of objtype) buf[N]
condition ok2add[]
condition ok2get

void maxlenbb.add(int bufnum, objtype obj){
    if(buf[bufnum].size()>=NELEM){
        ok2add[bufnum].wait()
    }
    buf[bufnum].insert(obj)
    ok2get.signal()
}

objtype maxlenbb.get(void){
    idx=getBiggest(buf)
    if(buf[idx].size()<=0){
        ok2get.wait()
        idx=getBiggest(buf)
    }
    objtype result=buf[idx].remove()
    ok2add[idx].signal()
    return result
}

int getBiggest(buffer buf[]){
    maxIdx=0
    for(i=0;i<N;i++){
        if(buf[i].size()>buf[maxIdx].size()){
            maxIdx=i
        }
    }
    return maxIdx
}
